API Docs

Organized API reference for Carsale Scraper, Real Footage Reels, and Global Make Model Video workflows inside Carbarn Operations Suite.

Carsale Scraper APIs

Base workflow path: /workflows/carbarn-vs-carsales/. API root alias is also available at /api/comparisons.

Method Path Purpose Notes
GET /workflows/carbarn-vs-carsales/api/comparisons/ Read comparison records and run results. Supports dashboard data loading and progress views.
POST /workflows/carbarn-vs-carsales/api/comparisons/ Start or submit comparison operations. Use JSON request body for comparison payload.
GET /api/comparisons Root-level API alias for comparisons endpoint. Useful for integrations using a stable global API prefix.

Real Footage Reels APIs

Base workflow path: /workflows/real-footage-reels/.

Method Path Purpose Notes
GET/workflows/real-footage-reels/api/metaRead app metadata and feature flags.Used during initial frontend boot.
GET/workflows/real-footage-reels/api/jobsList queued/running/completed jobs.Returns active IDs and worker mode.
POST/workflows/real-footage-reels/api/jobsCreate a new workflow job.Supports commands such as script draft and run resume flows.
POST/workflows/real-footage-reels/api/jobs/{job_id}/controlPause, resume, stop, or remove jobs.Action is passed in JSON body.
GET/workflows/real-footage-reels/api/runsList runs with status, stats, and latest job summary.Main runs grid data source.
GET/workflows/real-footage-reels/api/runs/{run_id}Get full run detail payload.Includes media URLs, scripts, plan, and pipeline state.
DELETE/workflows/real-footage-reels/api/runs/{run_id}Delete one run and related records.Legacy POST delete endpoint also exists.
POST/workflows/real-footage-reels/api/runs/{run_id}/prepareQueue prepare stage for an existing run.Shortcut to resume frame preparation.
POST/workflows/real-footage-reels/api/runs/{run_id}/prepare-analysisQueue prepare plus analysis pipeline.Supports auto-compose chaining.
POST/workflows/real-footage-reels/api/runs/{run_id}/identifyGenerate script options / identification stage.Routes into job command mapping in backend.
POST/workflows/real-footage-reels/api/runs/{run_id}/composeCompose full video with approved script.Requires `approvedScript` or `script` in body.
POST/workflows/real-footage-reels/api/runs/{run_id}/voiceover/draftRegenerate script options.Creates draft variants for approval.
POST/workflows/real-footage-reels/api/runs/{run_id}/voiceover/applyApply selected script and stitch voice-over.Persists applied script for run detail view.
POST/workflows/real-footage-reels/api/runs/{run_id}/thumbnailGenerate run thumbnail image.Accepts title, subtitle, and reference image payload.
GET/workflows/real-footage-reels/api/runs/{run_id}/asset?path=...Securely stream run assets.Supports range requests and optional attachment download flag.
POST/workflows/real-footage-reels/api/runs/trash-cleanupClean temporary and trash files.Requires confirmation flag in body or query.
GET/workflows/real-footage-reels/api/workflow-debugRead runtime diagnostics.Shows worker mode, node status, and key readiness.
GET/workflows/real-footage-reels/api/vehicle-inventory/statusRead inventory cache status.Returns counts and refresh state.
GET/workflows/real-footage-reels/api/vehicle-inventory/search?q=...&limit=20Search cached inventory.Used by studio inventory lookup UI.
POST/workflows/real-footage-reels/api/vehicle-inventory/refreshTrigger inventory refresh.Starts refresh and returns accepted response.
GET/workflows/real-footage-reels/api/public/stock-videos/endpointsList available public stock-video APIs.Discovery endpoint for external integrations.
GET/workflows/real-footage-reels/api/public/stock-videosList latest record per stock or by filter.Use `stockId` query for targeted lookup.
GET/workflows/real-footage-reels/api/public/stock-videos/{stock_id}Get latest record for one stock ID.Returns final reel and thumbnail metadata.
GET/workflows/real-footage-reels/api/public/stock-videos/{stock_id}/historyGet historical records for one stock ID.Supports `limit` query parameter.
